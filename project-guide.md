# Hyacintech STEM教育平台使用指南

## 项目概述

Hyacintech STEM教育平台是一个AI驱动的科学教育工具，旨在帮助学生系统地完成科学探究项目。平台基于上海市领先的《初中科学课程标准》，通过AI教师引导学生完成科学探究的六个关键阶段：选题定向、方案设计、过程执行、数据分析、成果成型和结果反思。

## 技术栈

- **前端框架**：Next.js (React)
- **样式**：Tailwind CSS
- **API**：Next.js API Routes
- **AI集成**：预留了与LLM模型（如OpenAI GPT-4o或DeepSeek）集成的接口

## 运行项目

### 环境要求

- Node.js 18.0及以上版本
- npm 或 yarn

### 步骤

1. **安装依赖**

```bash
cd ai-stem-platform/hyacinth
npm install
```

2. **设置环境变量**

创建`.env.local`文件，添加以下配置：

```
# 使用OpenAI (可选一种)
OPENAI_API_KEY=your_api_key_here

# 或使用DeepSeek (可选一种)
DEEPSEEK_API_KEY=your_api_key_here
DEEPSEEK_API_BASE=https://api.deepseek.com/v1
```

3. **启动开发服务器**

```bash
npm run dev
```

4. **访问平台**

打开浏览器，访问 [http://localhost:3000](http://localhost:3000)

## 核心功能介绍

### 1. 六阶段科学探究流程

平台将科学探究拆解为六个关键阶段，每个阶段有明确的学习目标和输出要求：

1. **选题定向**：引导学生将兴趣转化为可探究的科学问题
2. **方案设计**：协助学生设计实验方案，确定变量和材料
3. **过程执行**：指导学生收集数据和记录实验过程
4. **数据分析**：帮助学生分析数据并寻找规律
5. **成果成型**：引导学生整理发现并形成科学报告
6. **结果反思**：促进学生反思整个过程，总结经验教训

### 2. AI教师引导

AI教师通过结构化对话引导学生完成各个阶段，而不是直接给出答案。平台使用了精心设计的提示词工程，确保AI能够：

- 提供苏格拉底式引导，激发学生思考
- 针对每个阶段提供专业的科学指导
- 使用选项和结构化输出增强交互体验

### 3. 安全保障机制

平台内置了双层安全过滤系统：

- **前端过滤**：实时检测用户输入，阻止危险内容
- **后端验证**：API层面再次验证，确保实验安全性
- **AI内置规则**：提示词中嵌入安全原则，引导安全替代方案

## 常见问题

### Q: 为什么我收到安全提示？
A: 平台会过滤可能有安全隐患的实验材料或方法，比如强酸、高压电等。系统会提示更安全的替代方案。

### Q: 如何转到下一个阶段？
A: 当前阶段的核心任务完成后，AI会自动判断并允许你进入下一阶段。

### Q: 可以保存我的项目进度吗？
A: 当前版本暂不支持保存功能，建议手动记录重要内容。完整的数据保存功能将在后续版本中添加。
